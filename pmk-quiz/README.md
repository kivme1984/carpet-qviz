# PMK Quiz — версия для Tilda без ограничения по объёму кода

Проблема: полный квиз весил около 3.5 МБ из-за встроенных изображений base64, поэтому Tilda не принимает его в HTML-блок.

Решение: квиз вынесен в отдельную страницу `index.html`, изображения лежат в папке `assets/`, а в Tilda вставляется короткий iframe-код из `tilda-iframe-snippet.html`.

## Как поставить на сайт

1. Загрузите всю папку `pmk_quiz_external_embed` на GitHub Pages / сервер / хостинг.
2. Откройте получившуюся ссылку на `index.html`.
3. В файле `tilda-iframe-snippet.html` замените `QUIZ_URL` на эту ссылку.
4. В Tilda добавьте блок `T123 HTML` и вставьте только iframe-сниппет.
5. Опубликуйте страницу.

## Что сохранено

- Дизайн, анимации и логика квиза.
- Изображения без потери качества, но теперь они вынесены в файлы.
- Автовысота iframe через postMessage.
- На мобильной версии при переходе шага страница прокручивается к самому квизу.

## Размеры

- Полный старый HTML: 3647394 байт.
- Новый `index.html` без встроенных base64: 62969 байт.
- Tilda iframe-сниппет: 998 байт.
- Картинок вынесено: 11.
